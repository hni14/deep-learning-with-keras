# Deep learning with Keras
Kerasを中心に例題の解釈、ハンズオンを通じてディープラーニングに対する理解を深める。  

## システム要件

### オペレーティングシステム
- Ubuntu 16.04 or later (64-bit)
- macOS 10.12.6 (Sierra) or later (64-bit)
- Windows 7 or later (64-bit) (Python 3 only)

### python
- Python 2.7-3.6
